# small-o2o-compare

## 诞生背景

> 信创去O的过程中会经常需要进行数据库对象对比，其实用SQL和Excel也可以轻松实现。但是考虑程序也可以实现的，因此有了这个小工具。

## 元数据对比


目前支持的对比：

- [x] OB-ORACLE 和 ORALCE
- [x] ORACLE 和 ORALCE
- [x] OB-ORACLE 和 OB-ORALCE

后续计划支持

- [ ] OB-MYSQL 和 MYSQL
- [ ] MYSQL 和 MYSQL
- [ ] OB-MYSQL 和 OB-MYSQL
